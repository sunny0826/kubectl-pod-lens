## Requirements

- Kubernetes 1.19.0+
- Kubectl 1.18.0+
- Krew 0.4.0+

## Installation

```shell
kubectl krew install pod-lens
```

## help info

View the help information.

```shell
kubectl pod-lens -h
```


## That's it!

Congratulations! You've successfully install and run your `kubectl pod-lens`.
